## Output Files (Generated After Running)

When you run `python learn_sigma_beta_noisy_enkf.py`, the following files are created:

### Visualization Files
- **sigma_beta_convergence.png**
  - Two subplots showing parameter convergence
  - Subplot 1: Sigma with uncertainty bands
  - Subplot 2: Beta with uncertainty bands
  - Shows true value, initial guess, and ensemble mean

- **trajectory_fit.png**
  - Three subplots (one per state variable: x, y, z)
  - Compares ensemble mean predictions with observations
  - Shows how well the learned parameters fit

- **parameter_evolution.png**
  - Two subplots (sigma and beta)
  - Each shows evolution of all 50 ensemble members
  - Shows individual trajectories and ensemble mean
  - Visualizes ensemble spread reduction

- **trajectory_comparison.png**
  - Three subplots (x, y, z components)
  - Compares trajectories using:
    - True parameters
    - Initial parameter guesses
    - Final learned parameters
  - Shows improvement from learning

### Data Files
- **results.pt** (PyTorch file)
  - Complete results for post-analysis
  - Contains:
    - `X_hist`: State history (T, N, 3)
    - `theta_hist`: Parameter history (T, N, 3)
    - `sigma_hist_mean`: Sigma ensemble mean (T,)
    - `sigma_hist_std`: Sigma ensemble std (T,)
    - `beta_hist_mean`: Beta ensemble mean (T,)
    - `beta_hist_std`: Beta ensemble std (T,)
    - `true_params`: Ground truth parameters
    - `initial_params_mean`: Initial parameter guesses

### Log Files
- **learn_sigma_beta_noisy_enkf.log**
  - Generated by Hydra during execution
  - Contains console output
  - Helpful for debugging

- **.hydra/** (directory)
  - Hydra configuration files
  - Job metadata
  - Can usually be ignored
